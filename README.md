## 资源共享论坛

### 介绍
一个简单的论坛项目，使用github的api 进行登录

### 使用的工具
IDEA     
springboot
git
bootstrap
mysql/H2
myBatis
Flyway  Lombok shrio(可能会加入)

### 主要内容
用户登录  文章/图片上传 话题 热门用户等
